HTML Structure (Boilerplate)

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Page Title</title>
    <link rel="icon" type="image/x-icon" href="favicon.ico">
</head>
<body>
    
</body>
</html>
  • <!DOCTYPE html> : HTML5 declaration (always at top)
  • <html> : Root element
  • <head> : Metadata, title, links, favicon
  • <body> : All visible content (write all tags here)

Block vs Inline Elements

Block-level tags: Always start on a new line, take full width.
Examples: <div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>, <table>
Example:
Block 1
Block 2
Inline tags: Do not start on a new line, only take as much width as needed.
Examples: <span>, <a>, <img>, <strong>, <em>, <cite>
Example: Inline 1 Inline 2
MCQ: Block tags = new line, Inline = same line. div is block, span is inline.

Ordered List (ol) Attributes

Attributes:
  • reversed: Specifies that the list order should be reversed (9,8,7...)
  • type: The type attribute specifies the kind of marker to use in the list (letters or numbers).
  • start: The start attribute specifies the start value of the first list item in an ordered list.
Examples: 
<ol reversed>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

<ol type="A">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

<ol type="I">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

<ol start="50">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>
Output:
  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk
MCQ: type can be "1", "A", "a", "I", "i". start always uses a number.

Media Tags: Video, Audio, Iframe

Tag Purpose Sample Code Output Details
<video> Embeds a video file 
<video width="320" height="240" controls poster="poster.jpg">
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>
  • controls: shows play/pause etc.
  • width/height: set video size
  • Multiple <source> for formats
  • poster: Show image before play
  • autoplay, loop, muted supported
<audio> Embeds an audio file 
<audio controls autoplay loop>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio tag.
</audio>
  • controls: shows play/pause etc.
  • Use <source> for formats
  • autoplay, loop attributes available
<iframe> Embeds another webpage 
<iframe src="https://www.w3schools.com" width="300" height="150"></iframe>
  • src: URL of page to embed
  • width/height: set size
  • Can embed YouTube, Google Maps, etc.
  • allowfullscreen for full screen video
MCQ: <iframe> can embed any web page, YouTube, Maps, etc. Use title for accessibility.

MCQ & Output Tips

  • Semantic tags: <strong>, <em>, <abbr>, <cite>, <dfn>
  • Empty tags: <br>, <hr>, <img>, <input>
  • Block-level: <div>, <p>, <h1>-<h6>, <ul>, <ol>, <table>
  • Inline: <span>, <a>, <img>, <strong>, <em>, <cite>
  • Always write code inside <body> for visible output
  • MCQ: <head> is for meta, title, links; <body> is for visible content
  • MCQ: <input type="radio"> with same name = only one can be selected
  • MCQ: <label> is for accessibility, links to input by id
  • MCQ: <th> is table header, <td> is table data
  • MCQ: <dl> is description list, <dt> is term, <dd> is description
  • MCQ: <video> and <audio> need controls for user controls
  • MCQ: <iframe> can embed any web page, YouTube, Maps, etc.
  • MCQ: Use width and height to control size
  • MCQ: Multiple <source> tags = browser picks supported format
  • MCQ: Always use alt for images, title for iframes for accessibility
